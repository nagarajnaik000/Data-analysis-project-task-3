# Sales Data Analysis Using Pandas

## Overview

This project demonstrates how to perform data analysis using the Pandas library in Python. The dataset is loaded from a CSV file, cleaned, filtered, grouped, and analyzed to generate useful insights.

## Objectives

* Load data from a CSV file
* Clean and preprocess the dataset
* Filter specific records
* Group data for analysis
* Generate summary statistics
* Extract meaningful insights

## Technologies Used

* Python
* Pandas

## Project Structure

Sales-Data-Analysis/

├── sales_analysis.py

├── sales_data.csv

├── README.md

└── requirements.txt

## Dataset

The dataset contains product sales information including:

* Product Name
* Category
* Price
* Quantity Sold

## Features

### Data Loading

Loads CSV data using Pandas.

### Data Cleaning

* Removes duplicate records
* Handles missing values

### Data Filtering

Filters products based on categories.

### Data Grouping

Groups data by category and calculates total sales.

### Summary Statistics

Generates statistical information such as:

* Mean
* Count
* Minimum Value
* Maximum Value
* Standard Deviation

### Insights Generated

* Total sales by category
* Highest selling product
* Overall dataset statistics
